# The Four Types of Work

## Definition
A categorisation framework from *The Phoenix Project* stating that IT work falls into exactly four buckets:
1. **Business Projects:** Features the business wants.
2. **Internal IT Projects:** Infrastructure, refactoring, tech debt.
3. **Changes:** Updates generated by previous two.
4. **Unplanned Work:** Outages, incidents, "quick fixes" — the killer of capacity.
