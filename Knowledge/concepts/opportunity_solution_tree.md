# The Opportunity Solution Tree (OST)

## Definition
From Teresa Torres: A visual framework designed to help teams explicitly chart their path from a desired **Outcome** (business metric), to **Opportunities** (customer needs/pains), to **Solutions** (features), to **Experiments** (validation).

## Current Rule
- This is the anchor framework for the **AI in Scrum** pillar.
- Use it to enforce the rule: "AI is for Product Discovery, not training." 
- Teams fail when they use AI to generate code for bad ideas (Solutions layer). Teams succeed when they use AI to synthesize interviews and find patterns in the *Opportunities* layer.
